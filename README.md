# singapore.md

Singapore's README — a structured directory for AI agents and humans.

**Live site: [singapore.md](https://singapore.md)**

## What is this?

A single markdown file that serves as Singapore's "yellow pages" — government portals, industry data (AI, finance, green energy), and live data endpoints, structured for both AI agents and human readers.

## Content

- **Quick Facts** — timezone, currency, language, emergency numbers
- **Government Directory** — 16 ministries + key statutory boards
- **Industries — AI** — National AI Strategy 2.0, key programs and institutions
- **Industries — Finance** — MAS regulations, fintech ecosystem
- **Industries — Green Energy** — Green Plan 2030, targets
- **Live Data** — real-time weather, air quality, traffic, transport data

## For AI Agents

- Raw markdown: `https://singapore.md/index.md`
- Discovery protocol: `https://singapore.md/llms.txt`

## Development

```bash
npm install
npm run dev    # dev server at localhost:3456
npm run build  # build content/index.md → public/
```
